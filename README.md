# Утилита фильтрации содержимого файлов
    Программа на Java, которая читает один или несколько текстовых файлов и разделяет их содержимое на три категории:

    целые числа → integers.txt
    вещественные числа → floats.txt
    строки → strings.txt
    Поддерживает гибкую настройку вывода, режим добавления, краткую и полную статистику.

# Соответствие требованиям
    Чтение нескольких входных файлов по порядку
    Разделение данных по типам
    Имена по умолчанию: integers.txt, floats.txt, strings.txt
    Опция -o — задать путь вывода
    Опция -p — задать префикс имён файлов
    Опция -a — режим добавления (append)
    Файлы создаются только если есть данные
    
# Статистика:
    -s — краткая (только количество)
    -f — полная (мин/макс/сумма/среднее для чисел; длина строк)
    Обработка ошибок без остановки программы
    «Чистый» и понятный код
    
# Требования к окружению
    Компонент               Версия
    Язык                    Java
    Сборка                  Не обязательна (можно запускать напрямую), но поддерживается Gradle
    Сторонние библиотеки    Не используются — только стандартная библиотека Java (java.base)

# Инструкция по запуску
    Вариант 1: Без системы сборки (рекомендуется для проверки)
        1. Убедитесь, что установлен Java 11+:
            bash:
                java -version
                javac -version
        2. Скомпилируйте все файлы:
            bash:
                javac src/*.java
        3. Запустите программу с аргументами:
            bash:
                java -cp src FileFilter -f -a -p result_ input1.txt input2.txt
            Где:
                -f — полная статистика
                -a — добавлять в существующие файлы
                -p result_ — префикс имён выходных файлов
                input1.txt input2.txt — входные файлы
        4. Результаты появятся в текущей папке (или в указанной через -o).

    Вариант 2: С использованием Gradle (опционально)
        1. Создайте файл build.gradle в корне проекта:
            gradle:
                plugins {
                    id 'java'
                }

                java {
                    toolchain {
                        languageVersion = JavaLanguageVersion.of(17)
                    }
                }

                jar {
                    manifest {
                        attributes 'Main-Class': 'FileFilter'
                    }
                }
        2. Соберите проект:
            bash:
                gradle jar
        3. Запустите:
            bash:
                java -jar build/libs/filefilter.jar -s file1.txt            

    Пример использования

        Содержимое data.txt:
            Привет
            42
            3.1415
            Мир
            -100500
            1.23e-4

        Команда:
            bash:
                java -cp src FileFilter -f data.txt

        Результат в консоли:
            === Полная статистика ===
            Целые: 2, мин=-100500, макс=42, сумма=-100458, среднее=-50229.0000
            Вещественные: 2, мин=0.000123, макс=3.1415, сумма=3.141623, среднее=1.57081
            Строки: 2, мин. длина=4, макс. длина=6

        Файлы:
            integers.txt
            floats.txt
            strings.txt

# Особенности реализации
    Пустые строки игнорируются.
    Целые числа обрабатываются как long — поддерживаются очень большие значения (например, 1234567890123456789).
    Вещественные числа в научной нотации (1.5e-10) распознаются корректно.
    NaN, Infinity считаются строками (не записываются в floats.txt).
    Если во входных файлах нет данных определённого типа — соответствующий выходной файл не создаётся.
    При ошибке чтения одного файла программа продолжает обработку остальных.
